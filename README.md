# Task 3 – Earthquake Data Dashboard (Power BI)

## 📌 Objective

To create an interactive dashboard for analyzing global earthquake patterns using Power BI.

## 📊 Dashboard Features

* Total Earthquake Count
* Average Magnitude
* Maximum Magnitude
* Average Depth
* Total Affected Locations

## 📈 Visualizations

* Earthquake Map Visualization
* Magnitude Trend Over Time
* Depth vs Magnitude Scatter Plot
* Monthly Earthquake Distribution
* 
* Top 10 Earthquake Locations

## 🛠 Tools Used

* Power BI
* Python (for data preprocessing)
* Pandas

## 📂 Dataset

Earthquake dataset containing:

* Latitude
* Longitude
* Magnitude
* Depth
* Time information


## 📌 Insights

* Earthquake activity is concentrated in specific geographic regions.
* Magnitude distribution shows most events occur between magnitude 5 and 6.
* Depth has moderate correlation with magnitude.
